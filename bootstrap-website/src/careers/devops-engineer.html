About This Role

As a DevOps Engineer at DIDC, you'll be responsible for designing, implementing, and maintaining our cloud infrastructure that supports millions of data processing operations daily. You'll work with cutting-edge technologies including Kubernetes, Terraform, AWS services, and advanced monitoring solutions.

This is a hands-on role where you'll collaborate with development teams to implement CI/CD pipelines, ensure system reliability, and drive automation initiatives that improve our deployment velocity and system stability.

Impact You'll Make
  • Build infrastructure supporting 10M+ daily API requests
  • Achieve 99.9% uptime for critical business systems
  • Reduce deployment time from hours to minutes
  • Lead cloud cost optimization initiatives saving $500k+ annually
  • Mentor junior engineers and establish best practices
