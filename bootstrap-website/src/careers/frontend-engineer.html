About This Role

We're looking for a passionate Frontend Engineer to join our dynamic development team. You'll be responsible for building intuitive, performant, and accessible web applications that serve millions of users globally.

As a Frontend Engineer at DIDC, you'll work closely with our design and product teams to translate mockups and wireframes into pixel-perfect, interactive user interfaces. You'll be instrumental in establishing frontend best practices, optimizing performance, and ensuring our applications work seamlessly across all devices and browsers.

This role offers the opportunity to work with modern technologies including React, TypeScript, Next.js, and cutting-edge CSS frameworks. You'll contribute to our design system, implement new features, and help scale our frontend architecture to support rapid growth.

Key Responsibilities

React Application Development

Build and maintain scalable React applications using modern hooks, context, and state management patterns with Redux or Zustand.

Responsive Design Implementation

Create mobile-first, responsive designs that work flawlessly across all devices and screen sizes using CSS Grid, Flexbox, and modern CSS.

Design System Contribution

Help build and maintain our component library and design system, ensuring consistency and reusability across all products.

Performance Optimization

Optimize application performance through code splitting, lazy loading, bundle optimization, and implementing best practices for Core Web Vitals.

Testing & Quality Assurance

Write comprehensive tests using Jest, React Testing Library, and Cypress to ensure code quality and prevent regressions.
